Spring Boot + AngularJS

Ce projet est un exemple de projet Spring Boot + AngularJS.

Le projet est composé de 2 parties :

  • Une partie Java (Spring Boot) qui expose une API REST
  • Une partie AngularJS qui consomme cette API REST

Le projet est composé de 2 modules :

  • Un module Java (Spring Boot) qui expose une API REST
  • Un module AngularJS qui consomme cette API REST

Le module Java (Spring Boot) est composé de 2 packages :

  • Un package controller qui contient les contrôleurs REST
  • Un package model qui contient les modèles de données

Le module AngularJS est composé de 2 packages :

  • Un package app qui contient le module AngularJS
  • Un package app.controller qui contient les contrôleurs AngularJS
